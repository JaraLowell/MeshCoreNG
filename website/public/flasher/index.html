MeshCoreNG

LoRa mesh repeater firmware — dense-mesh telemetry • internet bridge • power saving
Use Chrome or Edge — Web Serial is not supported in Firefox or Safari.

1 Choose firmware type

2 Select your board

Select a firmware type first.

3 Connect & flash

Plug your board into USB, then press the button below.

Your browser does not support Web Serial. Use Chrome or Edge.

nRF52, RP2040 and STM32 boards cannot be flashed here. Download the .uf2, .zip or .hex from GitHub Releases.

4 First use

  1. Connect via USB serial at 115 200 baud (e.g. config.meshcore.io).
  2. For bridge firmware: set bridge.enabled on
  3. Check stats: get dense.stats
  4. Use any MeshCore client to connect: app.meshcore.nz
