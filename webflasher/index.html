MeshCoreNG

Web Flasher

Flash ESP32-based MeshCoreNG repeater firmware directly from Chrome or Edge. Pick your board, connect USB, then press install.

Important: Use this only with the matching board variant. The browser flasher supports ESP32-family devices. nRF52, RP2040 and STM32 boards still need their normal UF2, ZIP, HEX or platform-specific flashing flow.

What gets flashed?

These builds use the MeshCoreNG repeater firmware with dense-mesh telemetry and tuning. Default behavior stays compatible with existing MeshCore clients.

Browser support

Web flashing uses Web Serial, so it works best in Chrome or Edge on desktop. If the device is not detected, hold BOOT while connecting or use the board's normal flashing method.
